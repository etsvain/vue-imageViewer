# plugin-seeImage #

## 此插件依赖vue2.x+ 以及 element-ui 2.3+ ##

## 作者etsVain  邮箱535654314@qq.com ##

## 目录结构介绍 ##


## 使用步骤 ##
	1.将插件包下载
	2.将seeImage文件夹放到工程里
	3.在main中import  // import seeImage from './components/plugin/seeImage'
	4.Vue.use(seeImage);
	4.在需要点击查看的图片上加 @click="$seeImage"  //  <img src="/static/img/img.jpg" @click="$seeImage"/>

## 推荐使用yarn
    配置好了之后最好再install一下
